# Sample Output

These files are **illustrative examples only**. They show the shape of real output from
`python run.py security` and `python run.py export --overwrite`. Actual values (hashes,
timestamps, note paths) will differ when you run the pipeline against your own vault.

To generate real output, run:

```bash
python run.py export --overwrite   # produces dist/context-bundles/<id>/manifest.json
python run.py security             # prints security scan JSON to stdout
```
